1 3C分析

Customer（顧客） 必須
ターゲット顧客の特徴・ニーズ・行動パターン

ターゲット顧客の特徴
主なターゲットは20-30代の男女で、恋人・家族・友人へ誕生日や記念日、お礼の気持ちを込めてギフトを贈りたいユーザーです。職業は会社員や販売・サービス職など、スマートフォンで日常的に情報収集し、忙しい生活の中で効率よく買い物をしたい層を想定します。ライフスタイルとしてはSNS利用が日常化しており、商品の見た目やブランドの世界観を重視する傾向があります。

顧客のニーズや悩み
「花に詳しくないので何を選べばよいかわからない」「オンラインでは実物のイメージが湧きにくい」「ギフトなので失敗したくない」といった悩みが大きいです。また、配送日や鮮度、ラッピング、メッセージカード対応など、商品以外の安心感も重視されます。

顧客の行動パターン
情報収集はInstagram、TikTok、Pinterestなどのビジュアル重視SNSに加え、Google検索やECサイト内検索、レビュー確認を組み合わせる傾向があります。購買時は「誕生日 花 ギフト」「記念日 花 プレゼント」のようにシーン起点で検索し、写真、口コミ、価格、配送条件を比較しながらスマートフォン中心で購入する流れが想定されます。

市場のトレンド
花き市場では供給量の減少傾向が見られる一方で、EC利用やギフト需要は継続しており、オンライン上では「選びやすさ」と「ビジュアル訴求」が重要になっています。SNS時代においては、単に花の種類を並べるだけでなく、「どんな気持ち・どんなシーンに合うか」を明確に打ち出す提案型のUIが求められています。

Competitor（競合） 必須
競合サービスの特徴・強み・弱み

花キューピット
全国の加盟店ネットワークを活かした知名度の高いフラワーギフトサービスです。当日受付や全国配送への対応力が強みで、急ぎの贈り物ニーズに強い一方、サイト全体は用途提案よりも商品カテゴリや定番導線が中心で、若年層にとっては世界観がやや保守的に映る可能性があります。

LIFFT
洗練された世界観と高品質な花束表現を強みにしたオンラインフラワーショップです。見た目の美しさやギフト体験の質が高く、感度の高いユーザーに訴求しやすい一方、商品選択はデザイン重視で、花に詳しくないユーザーにとっては「自分に合う商品を選ぶ補助」がやや弱い可能性があります。

AND PLANTS
観葉植物を中心にしながら花束やギフト商品も展開し、ギフトラッピングやメッセージカード、法人利用などの対応が充実しています。植物と花をまとめて扱う幅広さが強みですが、Bloom Letter のように「贈るシーン別のフラワーギフト」に特化した訴求ではないため、花ギフト専門のわかりやすさでは差別化余地があります。

Company（自社） 必須
自分のサービスの強み・提供できる価値

提供できる価値
Bloom Letter は、花の知識が少ないユーザーでも「贈るシーン」から迷わず商品を選べる価値を提供します。誕生日、記念日、感謝などの目的別に商品を整理し、写真やシーンイメージを通じてオンラインでも購入後のイメージを持ちやすくすることで、ギフト選びの不安を減らします。

独自の強み
一般的な花ECサイトが花の種類や価格軸で商品を見せるのに対し、Bloom Letter は「誰に・どんな気持ちで贈るか」という文脈から選べる設計を採用している点が強みです。また、20-30代向けのナチュラルで見やすいUIにより、ギフトらしい雰囲気と選びやすさを両立できます。

活用できるリソース
フロントエンドでは HTML / CSS / JavaScript / Tailwind CSS、バックエンドでは Java / Spring Boot / MyBatis / MySQL を用いて、商品管理や一覧表示、詳細表示機能を実装できます。さらに、Figma によるUI設計、Git / GitHub による開発管理、AIツールによる企画・実装支援も活用可能です。

弱み・課題
新規サービスのため認知度や信頼性が低く、大手競合と比べて商品数、配送網、運営体制で不利です。また、花の品質管理、在庫管理、配送品質など、実運用面での課題も大きく、ギフト需要が季節イベントに左右されやすい点も課題といえます。

3 STP分析

Segmentation（市場細分化） 必須
市場をどのような基準で分類するか

地理的変数（地域・都市規模）
都市部在住でEC利用や即時性に慣れている層、地方在住で近隣店舗の選択肢が限られオンライン購入ニーズが高い層、配送対応エリア内の主要都市圏ユーザーに分けられます。

人口統計的変数（年齢・性別・職業・年収）
20代後半から30代前半を中心とした男女で、会社員、公務員、販売・サービス職など多忙でスマホ購買に慣れた層、中所得帯で価格と見た目のバランスを重視する層が想定されます。

心理的変数（ライフスタイル・価値観）
SNSを日常的に利用し、見た目や世界観を重視する人、相手に喜ばれるギフトを選びたい人、効率よく選びたいが安っぽく見える贈り物は避けたい人に分類できます。

行動変数（利用頻度・求めるベネフィット）
誕生日、記念日、母の日などイベント時に利用する低頻度ギフト購買層、普段は花を買わないが特別な機会に利用する層、選びやすさ・失敗しにくさ・見た目の良さ・配送の安心感を求める層に分けられます。

Targeting（ターゲット選定） 必須
どのセグメントを狙うか、その理由

選定したセグメント 1
20-30代の都市部在住会社員・公務員・販売サービス職で、恋人や配偶者、友人に誕生日・記念日ギフトを贈りたい層。

市場規模（推定）
総務省統計局の人口推計では、2025年9月1日現在の15-64歳人口は7348万3千人です。このうち Bloom Letter の中核ターゲットとなる20-39歳層は約4分の1から3分の1程度を占めると考えられるため、推定で約2,400万-2,700万人規模の母集団があると考えられます。この中でも、都市部在住かつオンラインでギフトを購入する層に絞ると、実用ターゲットは数百万人規模と推定できます。これは公的統計からの推定であり、厳密な市場実数ではありません。

選定理由
この層はスマートフォンでの情報収集とEC購入に慣れており、SNS経由で見た目や世界観に反応しやすい一方で、花の知識は必ずしも高くありません。そのため、「シーン別に選べる」「写真でイメージしやすい」という Bloom Letter の価値提案と最も相性が良いセグメントです。

このセグメントのニーズ
相手に喜ばれるギフトを失敗なく選びたい、短時間でセンスの良い商品を見つけたい、オンラインでも実物イメージを把握したい、配送やラッピングも含めて安心して購入したいというニーズがあります。

選定したセグメント 2
母の日やお礼、感謝の気持ちを伝える用途で家族向けギフトを探す20-30代後半の女性ユーザー。

市場規模（推定）
上記の20-39歳推定母集団のうち、家族向けギフト需要の高い女性層に絞ると、数百万人規模の候補市場があると考えられます。特にイベント需要が集中する時期には、短期間での購買意欲上昇が期待できます。

選定理由
家族向けギフトは「失礼がないこと」「気持ちが伝わること」が重視されやすく、Bloom Letter のシーン別提案と安心感訴求が効果を発揮しやすいためです。

このセグメントのニーズ
上品で失敗しにくい花を選びたい、用途に合う定番感とセンスを両立したい、メッセージ性やラッピング対応まで含めて安心して任せたい、というニーズがあります。

Positioning（ポジショニング） 必須
競合に対してどのような位置づけを取るか

差別化軸 1
商品起点ではなく「贈るシーン起点」で選べるわかりやすさ。

差別化軸 2
高級感や作家性だけでなく、ギフト初心者でも安心して選べる親しみやすさと実用性。

ポジショニングマップ上での位置
横軸を「商品軸で選ぶ - シーン軸で選ぶ」、縦軸を「高級・専門的 - 親しみやすく選びやすい」とすると、花キューピットは商品軸寄りかつ実用寄り、LIFFT は高級・世界観重視寄り、AND PLANTS は植物全般を扱う中間ポジションに位置づけられます。Bloom Letter は「シーン軸で選べる」かつ「親しみやすく選びやすい」領域に位置づけられます。

そのポジションを取る理由
大手と同じ土俵で商品数や配送網を競うのは難しい一方、花初心者向けのわかりやすさとギフト文脈に特化した導線は、比較的明確に差別化しやすい領域です。特に20-30代に対しては、専門性よりも「失敗しにくさ」「選びやすさ」「見た目の良さ」を同時に満たすポジションが有効だと考えられます。

4P・4C分析 必須

サービス概要
贈るシーン別に花を選べるフラワーギフトECサイト。

ターゲット
20-30代の男女。恋人・家族・友人へのギフトをオンラインで探したい人。特に花の知識が少なくても、相手に合った花を簡単に選びたい層。

4P（売り手視点）内容4C（買い手視点）内容
Product
（製品）		誕生日、記念日、感謝などシーン別に選べるフラワーギフトECサイト。商品写真、説明、利用シーン、ラッピング情報をわかりやすく掲載する。Customer Value
（顧客価値）		花に詳しくなくても、贈る目的に合った商品を迷わず選べる安心感を得られる。見た目と気持ちの両方が伝わるギフトを選びやすい。
Price
（価格）		ギフトとして選びやすい中価格帯を中心に設定し、価格に対する見た目の満足度と贈答価値のバランスを重視する。Cost
（顧客コスト）		予算内で相手に喜ばれる商品を選びやすく、比較検討にかかる時間や「選び方がわからない」心理的コストも下げられる。
Place
（流通）		Webサイト上で商品一覧、カテゴリ、詳細ページから購入まで完結できる導線を用意し、スマートフォン中心の利用を前提に設計する。Convenience
（利便性）		スマートフォンから短時間で探して購入でき、シーン別カテゴリや検索導線によって回遊しやすい。オンラインで完結できる手軽さがある。
Promotion
（販促）		SNS、ビジュアル訴求、ギフトシーン提案、季節イベント訴求を中心に集客を行う。誕生日、母の日、記念日など具体的な利用場面で接点をつくる。Communication
（対話）		写真、説明文、レビュー、FAQ、シーン別提案を通じて購入不安を減らし、「自分に合う商品が見つかる」感覚を提供する。

5 競合サイト分析

競合サイト 1 必須
サイト名・URL・スクリーンショット・分析

サイト名
花キューピット
URL: https://www.i879.com/

サイトの第一印象
大手サービスらしい安心感があり、母の日や誕生日などイベント需要に強い総合フラワーギフトサイトという印象です。情報量が多く、全国配送や用途別提案が前面に出ているため、実用性の高いサイトだと感じます。

良い点（デザイン・機能・コンテンツ）
イベント特集やランキング、用途別カテゴリが充実しており、ギフトを探す導線が豊富です。また、全国の花店ネットワークを活かした配送対応や当日受付など、機能面での安心感があります。定番ギフトを探すユーザーにとって必要な情報が揃っています。

改善できそうな点
情報量が多い分、若年層や花初心者には少し圧迫感があり、どこから選び始めるべきか迷う可能性があります。商品起点の導線が中心で、「気持ちから選ぶ」ような感覚的な選び方はやや弱い印象です。

自分のサイトに取り入れたい要素
イベントごとの特集展開、配送や注文締切のわかりやすい表示、ランキングや定番商品による比較のしやすさは取り入れたい要素です。一方で Bloom Letter では、それらをもっとシンプルなUIで見せる形に落とし込みます。

競合サイト 2 必須
サイト名・URL・スクリーンショット・分析

サイト名
LIFFT
URL: https://lifft.jp/

サイトの第一印象
上質で洗練されたブランドサイトという印象が強く、花を単なる商品ではなくライフスタイルや感性に結びついた存在として見せています。写真の見せ方や余白の使い方が美しく、世界観に引き込まれます。

良い点（デザイン・機能・コンテンツ）
ビジュアルの完成度が高く、花束そのものの美しさを強く感じさせるデザインです。ブランドストーリーや商品の背景が丁寧に伝えられており、単価が高くても納得感を持たせやすい構成になっています。写真・文章・世界観が一貫している点も大きな強みです。

改善できそうな点
デザイン性が高い反面、花初心者にとっては「どれを選べば自分に合うのか」がややわかりにくい可能性があります。用途別導線や比較しやすさよりもブランド表現が前面に出ているため、ギフト初心者には少しハードルがある印象です。

自分のサイトに取り入れたい要素
商品写真の見せ方、余白を活かした上質なレイアウト、ブランドの想いを伝えるストーリー性は取り入れたい要素です。Bloom Letter ではそこに「シーン別に選べるわかりやすさ」を加えて差別化します。

競合サイト 3 必須
サイト名・URL・スクリーンショット・分析

サイト名
AND PLANTS
URL: https://andplants.jp/

サイトの第一印象
植物全般をおしゃれに扱うライフスタイルECという印象で、親しみやすさと洗練さのバランスが良いサイトです。観葉植物の印象が強い一方で、ギフト需要にも対応していることが伝わります。

良い点（デザイン・機能・コンテンツ）
サイト全体が見やすく整理されており、カテゴリ回遊がしやすいです。商品写真が統一されていて比較しやすく、ラッピング、メッセージカード、法人向け対応など周辺情報もわかりやすく整備されています。初心者向けの読み物や説明が充実している点も良いところです。

改善できそうな点
観葉植物や植物全般を扱っている分、フラワーギフトに特化した専門感はやや薄く見えます。ギフトシーンに絞った導線はあるものの、「気持ち別・場面別に花を選ぶ」という特化度ではさらに明確化できそうです。

自分のサイトに取り入れたい要素
カテゴリの整理のしやすさ、初心者でも理解しやすい説明、ギフト対応情報の見せ方は取り入れたい要素です。Bloom Letter ではこれらを花ギフトに特化して、さらに目的別に選びやすく設計します。
