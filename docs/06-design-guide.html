1 デザインコンセプト

デザインキーワード 必須
デザインの方向性を表すキーワード3つとその説明

サービス概要
Bloom Letterは、大切な人に花で想いを届けるためのフラワーギフトサービスです。シーンや贈る相手に合わせて花を選びやすくし、安心してギフトを贈れる体験を提供します。

ターゲット
20代後半から30代前半の女性。スマートフォンを中心に情報収集を行い、誕生日、記念日、お祝い、母の日などのタイミングでギフトを探すことが多い層です。見た目の美しさだけでなく、選びやすさや安心感も重視します。

キーワード1：やさしさ
理由：花を贈る行為には、相手を想う気持ちを丁寧に届けたいという感情があるため、サービス全体にも柔らかく親しみやすい印象が必要である。
表現方法：白や淡いベージュ、やさしいグレーを基調にし、角を少し丸めたUIや余白のあるレイアウトで圧迫感を抑える。

キーワード2：上品さ
理由：フラワーギフトは特別感や贈答性が求められるため、華やかさがありながらも落ち着いた美しさを持たせることが重要である。
表現方法：装飾を増やしすぎず、整った文字組みや控えめな配色、洗練された余白設計で上質な印象をつくる。

キーワード3：わかりやすさ
理由：ギフト選びでは「何を選べばよいかわからない」という不安が起きやすいため、迷わず探せる導線設計が必要である。
表現方法：シーン別導線、商品一覧、CTAを整理して配置し、情報の優先順位がひと目で伝わるレイアウトにする。

2 カラーパレット

カラーパレット 必須
各色のHEXコード・使用場面を記載し、実際の色見本も表示してください
色の種類カラーコード色見本使用場面
メインカラー#D8BFAF見出し背景、やわらかい強調、ブランドの基調色
サブカラー#EDE6DDカード背景、セクション背景、区切り要素
アクセントカラー#8C6A56CTA、リンク、ボタン、重要導線
背景色#FAF7F2ページ全体の背景、余白部分
テキスト色#3F3A36本文テキスト、主要見出し
テキスト色（薄）#7A726B補助テキスト、注釈、キャプション
アイコン用アクセント#B8A9D9すみれモチーフのアイコン、控えめな装飾要素

選定理由
全体はベージュとグレージュを中心に構成し、やさしさと上品さを土台としている。メインカラーの #D8BFAF は思いやりや穏やかさを感じさせ、フラワーギフトに求められる温かみを表現できる。サブカラーの #EDE6DD と背景色の #FAF7F2 は、清潔感と安心感を与えながら商品写真を引き立てる役割を持つ。

アクセントカラーの #8C6A56 は、あたたかみのあるブラウンベージュで、信頼感と落ち着きを演出し、CTAや重要導線を視認しやすくする。テキスト色は、やわらかさを保ちながら十分な可読性を確保できる #3F3A36 と #7A726B を採用した。

アイコン用アクセントの #B8A9D9 は、薄紫のすみれをイメージした色で、やさしさと上品さを保ちながらブランドらしい印象を補強する。紫系には繊細さ、上質さ、落ち着きを連想させる心理効果があり、多用しないことで全体のトーンを崩さずに印象づけられる。

フォント設定 必須
要素フォント名サイズ太さ行間用途
h1（大見出し）Noto Sans JP40px7001.4トップのキャッチコピー、ページタイトル
h2（中見出し）Noto Sans JP28px7001.5セクション見出し
h3（小見出し）Noto Sans JP20px6001.6カード見出し、サブセクション
本文Noto Sans JP16px4001.9通常テキスト、説明文
キャプションNoto Sans JP13px4001.7補足情報、画像説明、注釈
ボタンNoto Sans JP14px5001.4ボタンテキスト、ナビゲーション

タイポグラフィ方針
フォントは Google Fonts で利用でき、日本語対応が安定している Noto Sans JP を採用する。やさしさ、上品さ、わかりやすさを同時に表現するため、可読性を優先しながらも、見出しには適度な強さを持たせて情報の階層を明確にする。

letter-spacing の推奨値
日本語は文字間を広げすぎると読みにくくなるため、全体を 0.02em〜0.05em の範囲に抑える。h1 は 0.02em、h2 と h3 は 0.03em、本文は 0.02em、キャプションは 0.04em、ボタンやナビゲーションは 0.05em を推奨する。

この設定により、本文は長文でも読みやすく、見出しはやわらかさを保ちながら整理された印象を与えられる。ECサイトとして必要な情報量に対応しつつ、ブランドの落ち着いた世界観も維持できる。

5 レイアウトルール

レイアウト基本ルール 必須
グリッド・余白・角丸・シャドウ等の数値ルール

デザインコンセプト
やさしさ / 上品さ / わかりやすさ

コンテンツ最大幅
1200px
余白を十分に確保しながら、商品カードや情報ブロックをゆったり見せられる幅とする。

グリッドシステム
PC：12カラム / ガター 24px
タブレット：8カラム / ガター 20px
スマートフォン：4カラム / ガター 16px
情報量の多いECサイトでも整理しやすく、下層ページや管理画面にも展開しやすい構成とする。

セクション間の余白
PC：120px
タブレット：88px
スマートフォン：64px
大きめの余白を確保することで、やさしさと上品さを保ちながら視線の切り替えをしやすくする。

要素間の余白
大：32px
中：24px
小：16px
極小：8px
8の倍数を基準に統一し、ページ全体のリズムと整然さをつくる。

角丸の数値
大きいカード・画像：20px
通常カード・入力欄：14px
ボタン：999px
小要素・タグ：10px
適度な丸みでやさしさを表現しつつ、丸くなりすぎない数値で上品さを保つ。

シャドウ設定
通常カード：box-shadow: 0 10px 24px rgba(140, 106, 86, 0.08);
ホバー時カード：box-shadow: 0 14px 32px rgba(140, 106, 86, 0.12);
ボタン：box-shadow: 0 6px 14px rgba(140, 106, 86, 0.18);
強すぎないやわらかな影で、ナチュラルな奥行きと上質感をつくる。

ブレークポイント
PC：1024px以上
タブレット：768px〜1023px
スマートフォン：767px以下
一般的で実装しやすく、今回のワイヤーフレーム構成に無理なく対応できる基準とする。

運用ルール
広い余白、12/8/4カラム、8の倍数ベースのスペーシングを共通ルールとし、どのページでも統一感のあるレイアウトを維持する。
